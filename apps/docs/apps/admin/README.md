# Admin Dashboard

The Admin Dashboard is a web-based management interface for AuthHero. It allows administrators to manage tenants, applications, connections, domains, users and branding through a user-friendly interface.

## Features

- Comprehensive tenant management
- Application configuration
- Connection setup and management
- Domain configuration
- User management
- Log viewing and analysis
- Whitelabel branding (app name, logo, favicon)

## Overview

The dashboard is a Vite + React 19 SPA built with [shadcn/ui](https://ui.shadcn.com/) and `ra-core`, communicating with your AuthHero backend via the management API.

For setup instructions, see the [Installation](./installation.md) page. For usage details, see the [Usage](./usage.md) page.
